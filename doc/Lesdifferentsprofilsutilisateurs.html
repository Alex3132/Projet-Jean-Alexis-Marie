Les différents profils utilisateurs

Spécifications ››


Il y a 3 types d'utilisateurs :


  1. admin => peut modifier l'ensemble des champs qui ne sont pas présents dans la table "readonly" (cf readonly).
  2. edit => ne peut peut modifier que les champs "code postal" et "population" de la table ville (cf villes).
  3. anonyme => ne peut accéder qu'aux pages :
    1. Page d'accueil,
    2. Page de login
    3. Page de recherche


L'inscription sur le site ne fournit pas de profil, seul le super administrateur peut changer le profil d'un utilisateur, un formulaire de demande de changement de status sera fourni ultérieurement.

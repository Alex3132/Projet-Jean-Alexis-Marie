Spécifications ››



Lors de l'arrivée sur le site on se retrouve sur la Page d'accueil.

L'utilisateur n'est pas identifié et le menu Maintenance est caché.


L'utilisateur peut s'identifier par le biais du menu identification s'il est déjà inscrit, sinon il peut s'inscrire par le même menu qui mène sur la page : Page de login.


Une fois identifié et si son profil est  "edit"  ou "admin", alors le menu de maintenance est accessible.


Le menu Recherche est toujours visible et mène sur la Page de recherche.
